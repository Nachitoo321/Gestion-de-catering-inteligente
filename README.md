# Sistema de Gestión de Catering 🍽️

Sistema desarrollado en Java que simula la gestión de un servicio de catering.

## Funcionalidades
- Registrar producción de platos por chef
- Ver total de ventas generadas
- Ver bonos por producción (5% sobre ventas)
- Ver plato más popular del mes
- Ver chef del mes

## Tecnologías
- Java
- Arrays y matrices
- Descomposición modular (métodos)
